To run project:

1. The trivia-api folder contains the API writen in NodeJS which needs to be hosted on a server first.
2. The database was hosted on MongoDB, however, it isn't anymore. So, to run the project the database would need to be remade and rehosted.
2. Open terminal inside of dubrovnik-trivia-app
3. Execute command 'npm install'
4. Execute command 'npm start'
5. Add '/admin' on top of the home url to access the admin panel
